# Проект анализа бизнеса (прототип)

## Описание
Этот проект выполняет анализ бизнес-данных по различным регионам и генерирует отчеты для сравнения и обзора.

## Структура проекта
- `main_pro.py` - основной скрипт для анализа данных
- `regions.csv` - данные о регионах
- `businesses.csv` - данные о бизнесах
- `assumptions.csv` - предположения/допущения
- `report_*.txt` - сгенерированные отчеты

## Использование
Запустите основной скрипт для генерации отчетов:
```bash
python main_pro.py
```

## Отчеты
Проект генерирует следующие типы отчетов:
- Отчеты по отдельным регионам (`report_single_*.txt`)
- Сравнительные отчеты между регионами (`report_compare_*.txt`)
- Общий обзор всех регионов (`report_overview_all.txt`)
